# Arquivos estáticos

Coloque aqui os arquivos da identidade visual:

| Arquivo     | Descrição              | Formato recomendado |
|-------------|------------------------|---------------------|
| `icon.ico`  | Favicon (ícone da aba) | .ico (16x16 ou 32x32) |
| `logo.png`  | Logo no header         | .png ou .svg        |

Se usar `logo.svg` em vez de `logo.png`, atualize o caminho em `app/(panel)/layout.tsx`.
